# CARA: Cloud Requirements Analyzer Agent

This repository contains the source code, experimental results, and figures for the paper:

**"CARA: A Bilingual Agentic AI Framework for Cloud Requirements Analysis"**

## Contents
- `CARA_agent_final.py` - The complete CARA agent code.
- `CARA_Final_Report.json` - Full experimental results (15 references, 3 test settings).
- `CARA_Experimental_Results.csv` - Summary table of results.
- `figure1_results.png` - Figure 1: Distribution of FR and NFR requirements.

## How to Reproduce
1. Run `CARA_agent_final.py` in any Python environment (Google Colab recommended).
2. The agent will extract and classify requirements from input text.
3. See the paper for full methodology and discussion.
